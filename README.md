# Задание 1



## Задача 1



### Текст задачи



На основе списка, содержащего числа от 1 до 7, получить список из

соответствующих названий дней недели. 



### Алгоритм решения



1) Попадаем в функцию main

2) Вызываем функцию readList

3) В функции readList запрашиваем у пользователя номер дня недели, считываем строку с консоли. Проверяем пустая ли строка или равна null, если да возвращаем пустой список, если нет пытаемся преобразатиь строку в число. Если преобразование успешно и число от 1 до 7, то рекурсивно вызываем readList для получения остатка списка, добавляем текущее число в начала списка и возвращаем результат. Если преобразование не успешно или число вне диапазона, выводи ошибку и повторяем ввод (вызываем readList снова)
   
4) Получаем результат numbers от функции readList
   
5) Проверяем пустой ли список numbers, если список пуст выводим "Список пуст", если нет, то преобразуем каждый элемент списка в название дня через List.map getDayName, выводим название в каждой новой строке 

6) Завершаем программу

# Задание 2



## Задача 1



### Текст задачи



Дан список, содержащий натуральные числа. Сколько из них содержит в своей записи k цифр 



### Алгоритм решения



1) Попадаем в функцию main

2) Вызываем функцию readNumbers для ввода списка чисел, запрашиваем ввод числа с клавиатуры, проверяем введено ли "стоп", если да, то возвращаем накопленный список чисел. Если введено не "стоп", проверяем ли является ввод целым числом, если нет выводим ошибку: введите целое число, и повторяем ввод, если является добавляется число в спислк и продолжаем ввод, возвращаем список чисел в main.

3) Запрашиваем ввод числа k(кол-во цифр), проверяем является ли ввод целым полож. числом, если нет выводим ошибку: введите положительное число

4) Если k введено корректно, вызываем функцию countNumbersWithKDigits

5) Использует List.fold для подсчета, для каждого числа определяем кол-во цифр, если число = 0, то кол-во цифр = 1, иначе берем модуль числа, преобразуем в строку и считаем длину. Елс кол-во цифр равно k, то увеличиваем счетчиак на 1. 

6) Возвращает итоговое кол-во чисел

7) Выводим результат на экран

8) Завершаем программу
